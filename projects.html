The process of enhancing the “List This!” app was challenging yet served its purpose as a learning experience in a myriad of ways. Transitioning the app from a SQLite database to a Firebase platform provided me with several benefits such as understanding real-time data synchronization. Firebase offers real-time data synchronization, which means any changes made to the data are immediately updated on all connected devices. Firebase is a cloud-based platform, which means it can scale automatically to handle increasing traffic and data storage needs. This can save time and resources on managing infrastructure. In my research of security and authentication I also found that Firebase offers built-in authentication and security features, such as email and password authentication, social media logins, and server-side security rules. This can help secure the app and protect user data. An understanding of analytics and performance was also realized as monitoring tools that can track user engagement, app usage, and performance issues make data-driven decisions to improve the app. More complex features and options were also available through cloud functions as Firebase allows for server-side code without necessarily managing server infrastructure.
