The process of enhancing the “List This!” app was challenging yet served its purpose as a learning experience in a myriad of ways. I have transitioned the “List This! Lite” app from a SQLite database to a Firebase platform. This provided me with several benefits, such as understanding real-time data synchronization. Firebase offers real-time data synchronization, which means any changes made to the data are immediately updated on all connected devices. Firebase is a cloud-based platform, which can scale automatically to handle increasing traffic and data storage needs. This can save time and resources on managing infrastructure. My app was enhanced in a way where monitoring tools can track user engagement, app usage, and performance issues, making data-driven decisions to improve the app. More complex features and options were also available through cloud functions as Firebase allows for server-side code without necessarily managing server infrastructure.
